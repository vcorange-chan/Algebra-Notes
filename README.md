# Algebra-Notes

A personal, long-term collection of my algebra notes, ideas, and conversations with Claude.

## What's in here

This repo is a working notebook — a place to capture material as I learn, not a polished textbook. Expect rough edges. Things will be reorganized over time.

Current contents may include:

- **Notes** — markdown write-ups of concepts, proofs, and definitions I'm working through.
- **Ideas** — half-formed thoughts, conjectures, questions to come back to.
- **Chats** — transcripts and excerpts of conversations with Claude about algebra.
- **Exercises** — problems I'm solving, with my attempts and solutions.
- **Techniques** — reusable tricks, patterns, and problem-solving approaches.

## How I use this repo

The priority right now is **capture over organization**. I'll collect material first and sort it into a proper structure later. If something is useful, it belongs here — even if it's not yet in the "right" place.

## Status

Work in progress, indefinitely.
